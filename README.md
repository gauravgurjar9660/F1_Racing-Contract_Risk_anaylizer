# F1 Racing Contract Risk Analyzer 🏎️🏁

A data-driven decision support tool designed to analyze F1 circuit contracts. This application evaluates contract durations and status, helping stakeholders identify high-risk expiring contracts to ensure long-term racing calendar stability.

## 🚀 Key Highlights
* **Risk Intelligence:** Automatically identifies and highlights circuits with contracts nearing expiration.
* **Streamlined Analysis:** Translates complex contract timelines into clear, actionable insights.
* **Decision Support:** Provides a high-level view of contract health, assisting in strategic planning for the racing calendar.

## 🛠 Tech Stack
* **Language:** Python
* **Data Processing:** Pandas, NumPy
* **Frontend:** Streamlit
* **Deployment:** Streamlit Cloud

## 🧠 What I Learned
* **Data-Driven Decision Making:** Gained insights into how raw contractual data can be transformed into strategic risk metrics.
* **Pipeline Automation:** Developed an end-to-end processing pipeline that automatically classifies contracts based on their maturity (Active/Expired/Expiring).
* **Business Logic Implementation:** Applied time-series logic to calculate remaining contract durations and flag potential "risk" windows.

## 📊 How It Works
1. **Data Input:** Processes structured contract datasets containing circuit names and agreement dates.
2. **Analysis:** Calculates remaining time and categorizes circuits into risk tiers.
3. **Visualization:** Displays clear status flags for immediate awareness of contract stability.
